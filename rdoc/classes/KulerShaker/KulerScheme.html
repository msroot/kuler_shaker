Class KulerShaker::KulerScheme
In: lib/kuler_shaker/kuler_scheme.rb
Parent: Object

Contains the data objects necessary to work with a single Kuler scheme. See the Adobe API reference for more info (not that there‘s a whole lot! Grr!)

Attributes

author_id  [RW]  the ID of the author for this swatch
author_name  [RW]  the name of the author for this swatch
created_at  [RW]  date this scheme was created
download_count  [RW]  the download count for this scheme
edited_at  [RW]  date this scehme was edited
image  [RW]  the url of the image used by this theme (the adobe swatch .png)
rating  [RW]  the rating for this scheme
swatches  [RW]  an array of *KulerShaker::KulerSwatch* objects, representing the colours that make up this scheme
tags  [RW]  an array of tags associated to this scheme
title  [RW]  the title of this scheme
